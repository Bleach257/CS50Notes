### How the Web Works

```
Browser  →  DNS lookup  →  IP address
         →  TCP/IP connection
         →  HTTP request  →  Server
                          ←  HTTP response (HTML/CSS/JS)
         →  Render page
```

### HTML Structure

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <h1>Heading</h1>
    <p>Paragraph</p>
  </body>
</html>
```
