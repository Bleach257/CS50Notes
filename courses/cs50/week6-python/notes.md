### C vs Python Quick Comparison

| Feature | C | Python |
|---------|---|--------|
| Compilation | Compiled | Interpreted |
| Type system | Static | Dynamic |
| Memory | Manual (`malloc`/`free`) | Automatic (garbage collection) |
| String type | `char *` / `string` | `str` |
| Print | `printf("Hello\n")` | `print("Hello")` |
| Conditions | `if (x > 0)` | `if x > 0:` |
| Loops | `for (int i = 0; i < n; i++)` | `for i in range(n):` |
